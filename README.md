# multitasking-scheduling-algorithm-simulation-system
multitasking scheduling algorithm simulation system

# raylib:
 ```
 mkdir build_raylib
 cd build_raylib
 cmake ..
 make
 ```
 in case of `-bash: ./build.sh: \bin\sh^M: bad interpreter: No such file or directory` error run this :
```
sed -i -e 's/\r$//' build.sh
```
